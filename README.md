# CASL2サンプルコード
## コンパイル
node-casl2を使用
[node-casl2](https://github.com/node-casl2-comet2/node-casl2)

```sh
node-casl2 test.cas -o 
```

## CLI実行
node-comet2を使用
[node-comet2](https://github.com/node-casl2-comet2/node-comet2)

```sh
node-comet2 -i test.com
```
